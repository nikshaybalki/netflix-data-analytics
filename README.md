# 🍿 Netflix Data Analytics: Exploratory Data Analysis (EDA)

## 📝 Project Overview
This project focuses on performing an in-depth Exploratory Data Analysis (EDA) on the Netflix catalog dataset. By using Python to clean an uncurated dataset of over 8,800 entries and plot the results, this analysis uncovers the underlying business trends driving Netflix's platform strategy.

## 📂 Dataset
* **Source:** Netflix Movies and TV Shows Dataset 
* **Description:** The dataset contains **8,807** rows and 12 columns, including detailed information on movies and TV shows such as the director, cast, country of origin, release year, rating, and duration.
* **Main File:** `project.ipynb`

## 🛠️ Tools & Libraries Used
This project was built in **VS Code** using Python. The following libraries were utilized:
* **Pandas:** For data manipulation and cleaning the uncurated dataset.
* **Matplotlib & Seaborn:** For generating data visualizations and plotting trends.
* **NumPy:** For numerical computations and handling missing data.
* **Jupyter Notebook:** For interactive analysis.

## 💡 Project Insights: The Story of the Data
Here are the major exploratory data analysis findings and the business trends they reveal:

1. **Catalog Composition (Bar Chart):** Despite the cultural rise of binge-worthy series, standalone Movies significantly outnumber TV Shows, remaining the core pillar of Netflix's inventory structure.
2. **Demographic Targeting (Pie Chart):** The library heavily favors TV-MA and TV-14 ratings over family-friendly content, proving a strategic focus on mature and young-adult viewership segments.
3. **Geographical Density (Horizontal Bar Chart):** The United States sits uncontested as the primary content producer, while India and the United Kingdom firmly hold the secondary tier of dense media generation.
4. **Continuous Feature Distribution (Histogram):** Movie runtimes sharply normalize and peak at the 90-to-100-minute mark, indicating calculated tailoring to modern consumer attention spans.
5. **Temporal Cluster Analysis (Scatter Plot):** Plotting timelines reveals an exponential content spike over the last decade, visually capturing the exact era Netflix shifted from a licensing platform to an original production powerhouse.
